# Student Report Card System (C++)

## 📌 About the Project

This project is a **Student Report Card System** built using **C++ and Object-Oriented Programming**.
The program collects student information such as name, roll number, and marks, then calculates the **average marks** and assigns a **grade** based on the result.

The goal of this project is to practice **OOP concepts, arrays, and conditional logic**.

---

## 🛠️ Technologies Used

* C++
* Object-Oriented Programming (Classes & Objects)
* Arrays
* Conditional Statements (if
* Console Input/Output

---

## ✨ Features

* Stores **student name and roll number**
* Accepts **multiple subject marks using arrays**
* Calculates **average marks**
* Assigns **grade based on performance**
* Displays a complete **student report card**

---

## ⚙️ How It Works

1. User enters student name and roll number
2. Program receives marks for different subjects
3. Marks are stored in an **array**
4. The program calculates the **average score**
5. Grade is assigned using **if–else conditions**
6. The final **report card is displayed**
